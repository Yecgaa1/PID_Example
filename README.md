## 概念详解
### 积分分离（解决启动和机动时的偏差值太大）

当被控量与设定值偏差较大时，取消积分作用; 当被控量接近给定值时，引入积分控制

### 抗饱和（解决死区）

在计算u(k)时，首先判断上一时刻的控制量u(k-1)是否已经超出了极限范围： 如果u(k-1)>umax，则只累加负偏差; 如果u(k-1)<umin，则只累加正偏差。从而避免控制量长时间停留在饱和区。

### 梯形积分

提高积分项运算精度，把矩形积分改为梯形积分

### 变积分

系统偏差大时，积分作用应该减弱甚至是全无，而在偏差小时，则应该加强
